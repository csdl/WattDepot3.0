  • GET /wattdepot/locations Get a full list of the defined Locations.
  • GET /wattdepot/location/{location_id} Get the Location with the given name.
  • POST /wattdepot/locations Create a new Location. It must have a unique name.
  • DEL /wattdepot/locations/{location_id} Remove a Location.

Get a full list of defined Locations.

GET /wattdepot/location/{location_id}

http://server.wattdepot.org/wattdepot/location/ilima-6th Response 
              
{
   "description": "Hale Aloha Ilima residence hall 6th floor",
   "altitude": 30,
   "longitude": -157.812727,
   "latitude": 21.294642,
   "uniqueId": "ilima-6th"
}

Get the Location with the given location_id.

PUT /wattdepot/location/

curl -F '{"description":"Hale Aloha Ilima residence hall 6th floor","altitude":30,"longitude":-157.812727,"latitude":21.294642,"uniqueId":"loc1"}' http://server.wattdepot.org/wattdepot/location/ Response 
              
{
   "status": 200,
   "message": "OK"
}

Add the Location. Will return the status. Error status if duplicate unique id.

DEL /wattdepot/location/{location_id}

curl -X DELETE http://server.wattdepot.org/wattdepot/location/ilima-6th Response 
              
{
   "status": 200,
   "message": "OK"
}

Deletes the Location with the given location_id. Will return the status. Error status if duplicate unique id.
