  • GET /wattdepot/sensors Get a full list of the defined Sensors.
  • GET /wattdepot/sensor/{sensor_id} Get the Sensor with the given name.
  • POST /wattdepot/sensors Create a new Sensor. It must have a unique name.
  • DEL /wattdepot/sensors/{sensor_id} Remove a Sensor.

GET /wattdepot/sensors

http://server.wattdepot.org/wattdepot/sensors Response

   {
      "uniqueId": "ilima-6-telco",

Get a full list of defined Sensors.

GET /wattdepot/sensor/{sensor_id}

http://server.wattdepot.org/wattdepot/sensor/ilima-6-telco Response 
              
{
   "uniqueId": "ilima-6-telco",
   "uri": "http://foo.com/ilima-6-telco",
   "location": "ilima-6th",
   "model": "eguage1.0",
   []
}

Get the Sensor with the given sensor_id.

PUT /wattdepot/sensor/

curl -F '{uniqueId: "sensor1", uri: "http://foo.com", location: "loc2", model: "sm1", []}' http://server.wattdepot.org/wattdepot/sensor/ Response 
              
{
   "status": 200,
   "message": "OK"
}

Add the Sensor. Will return the status. Error status if duplicate unique id.

DEL /wattdepot/sensor/{sensor_id}

curl -X DELETE http://server.wattdepot.org/wattdepot/sensor/ilima-6th Response 
              
{
   "status": 200,
   "message": "OK"
}

Deletes the Sensor with the given sensor_id. Will return the status. Error status if duplicate unique id.
