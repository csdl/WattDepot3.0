Get the full Sensor with the given sensor_id.

PUT /wattdepot/{group_id}/sensor/

curl -F '{"id": "sensor1", "uri" "http://foo.com", "location": {id :"loc2", "latitude": 21.294642, "longitude": -157.812727, "altitude": 40.0, "description": "40.0"}, "model": {id :"sm1", "protocol": "Hammer", "type": "hammer", "version": "1.0"}, []}' http://server.wattdepot.org/wattdepot/{group_id}/sensor/ 
              
{
   "status": 200,
   "message": "OK"
}

Add the Sensor. Will return the status. Error status if duplicate unique id.

DEL /wattdepot/{group_id}/sensor/{sensor_id}

curl -X DELETE http://server.wattdepot.org/wattdepot/{group_id}/sensor/ilima-6th 
              
{
   "status": 200,
   "message": "OK"
}

Deletes the Sensor with the given sensor_id. Will return the status. Error status if sensor_id is not defined.
