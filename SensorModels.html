Get a full list of defined Sensor Models.

GET /wattdepot/sensormodel/{sensormodel_id}

http://server.wattdepot.org/wattdepot/sensormodel/egauge1.0 Response 
              
{
   "id": "egauge1.0",
   "protocol": "eGauge",
   "type": "EGAUGE",
   "version": "1.0"
}

Get the Sensor Model with the given sensormodel_id.

PUT /wattdepot/sensormodel/

curl -F '{id :"egauge1.0", "protocol": "eGauge", "type": "EGAUGE", "version": "1.0"}' http://server.wattdepot.org/wattdepot/sensormodel/ Response 
              
{
   "status": 200,
   "message": "OK"
}

Add the Sensor Model. Will return the status. Error status if duplicate unique id.

DEL /wattdepot/sensormodel/{sensormodel_id}

curl -X DELETE http://server.wattdepot.org/wattdepot/sensormodel/egauge1.0 Response 
              
{
   "status": 200,
   "message": "OK"
}

Deletes the Sensor Model with the given sensormodel_id. Will return the status. Error status if duplicate unique id.
