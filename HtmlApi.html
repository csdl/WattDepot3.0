WattDepot3 HTML API

GET
/wattdepot/locations
Returns all the known locations.
GET
/wattdepot/location/{location_name}
Returns the location with the given name.
PUT
/wattdepot/location/{location_name}
Adds the location with the given name.
DELETE
/wattdepot/location/{location_name}
Removes the location with the given name.
