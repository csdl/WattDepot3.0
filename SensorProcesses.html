Get the Sensor Process with the given sensorprocess_id.

PUT /wattdepot/{group_id}/sensorprocess/

curl -F '{id :"egauge1-fast", "sensor": {id: "sensor1", uri: "http://foo.com", location: {id :"loc2", "latitude": 21.294642, "longitude": -157.812727, "altitude": 40.0, "description": "40.0"}, model: {id :"egauge1", "protocol": "eGauge", "type": "EGAUGE", "version": "1.0"}, []}, "pollingInterval": 10}' http://server.wattdepot.org/wattdepot/{group_id}/sensorprocess/ 
              
{
   "status": 200,
   "message": "OK"
}

Add the Sensor Process. Will return the status. Error status if duplicate unique id.

DEL /wattdepot/{group_id}/sensorprocess/{sensorprocess_id}

curl -X DELETE http://server.wattdepot.org/wattdepot/{group_id}/sensorprocess/egauge1.0 
              
{
   "status": 200,
   "message": "OK"
}

Deletes the Sensor Model with the given sensorprocess_id. Will return the status. Error status if sensorprocess_id is not defined.
