Get the list of Sensors that sent measurements to the WattDepository with the given wattdepository_id.

GET /wattdepot/wattdepositorie/{wattdepository_id}/sensors?short=true

http://server.wattdepot.org/wattdepot/wattdepositorie/ilima-energy1/sensors?short=true Response 
              
[
   "ilima-6-telco",
   "ilima-6-elect"
]

Get the list of Sensors that sent measurements to the WattDepository with the given wattdepository_id in shortened format. Shortend format uses sensor ids instead of full sensors.

GET /wattdepot/wattdepositorie/{wattdepository_id}/{sensor_id}/value/{timestamp}

http://server.wattdepot.org/wattdepot/wattdepositorie/ilima-energy1/ilima-6-telco/value/2013-07-28T09:00:00.000-10:00 Response 
              
{
  "sensorId": "ilima-6-telco",
  "value": 130531.388,
  "measurementType": "energy"
}

Get measurement value for the given sensor at the given time. This is most likely an interpolated value.

The request will return an error if the timestamp is less than the eariliest measurement time or greater than the latest measurement time.

GET /wattdepot/wattdepositorie/{wattdepository_id}/{sensor_id}/value?start={timestamp},end={timestamp}

http://server.wattdepot.org/wattdepot/wattdepositorie/ilima-energy1/ilima-6-telco/value?start=2013-07-28T09:00:00.000-10:00,end=2013-07-29T09:00:00.000-10:00 Response 
              
{
  "sensorId": "ilima-6-telco",
  "value": 372.84,
  "measurementType": "energy"
}

Get measurement value for the given sensor over the time period. The value at start is subtracted from the value at end.

The request will return an error if the start or end timestamp is less than the eariliest measurement time or greater than the latest measurement time.

PUT /wattdepot/wattdepositorie/

curl -F '{"id": "wattdepositorie1", "name": "Ilima energy consumption", "measurementType": "energy"}' http://server.wattdepot.org/wattdepot/wattdepositorie/ Response 
              
{
   "status": 200,
   "message": "OK"
}

Add the WattDepository. Will return the status. Error status if duplicate unique id.

PUT /wattdepot/wattdepositorie/{wattdepository_id}

curl -F '{"sensorId": "ilima-6th-telco", "value": 793l4.54, "timestamp": "2013-07-28T09:02:00.000-10:00", "measurementType": "energy"}' http://server.wattdepot.org/wattdepot/wattdepositorie/ilima-energy1 Response 
              
{
   "status": 200,
   "message": "OK"
}

Add the measurement to the given WattDepository. Will return the status. Error status if measurement type of the measurement doesn't match the WattDepositories measurement type.

DEL /wattdepot/wattdepositorie/{wattdepository_id}

curl -X DELETE http://server.wattdepot.org/wattdepot/wattdepositorie/ilima-energy1 Response 
              
{
   "status": 200,
   "message": "OK"
}

Deletes the Sensor with the given wattdepository_id. Will return the status. Error status if duplicate unique id.
